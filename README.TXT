本项目简介
	该项目为提供各各平台通用基础数据服务平台，所有基础服务数据（用户的基本信息）提供给各各平台
	进行基础数据采集服务。
本项目技术架构设计
	采用主流的REST面向服务面向组件的架构设计思想下采用SpringMVC的REST技术实现规范的实现。
	SpringMVC+Hibernate+DB+Maven+Git(版本控制
